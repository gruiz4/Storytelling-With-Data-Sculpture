# Storytelling-With-Data-Sculpture
Circuit Python code for ESP32 based sculpture that uses Servos and Neopixel LEDs to display live(ish) data on the relative amount and source of energy production in the United States. 

Live Energy Data is sourced using an API provided by electricitymaps.com. 
A cool map of the world's energy production can be found on their website as well: https://app.electricitymaps.com/map/live/fifteen_minutes. 

For more details see API Documentation: https://app.electricitymaps.com/developer-hub/api/reference#electricity-mix-latest
